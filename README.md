# MERN Auth Dashboard 

A simple MERN authentication project with user signup, login, and a protected dashboard.

## Features
- Signup & Login
- JWT Authentication
- Protected Dashboard
- Token stored in LocalStorage
- Works on Local & Production

## Tech Stack
- React (Vite)
- Axios
- React Router DOM
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT

### Run Locally

### Frontend
```
cd frontend
npm install
npm run dev
```

### Backend
```
cd backend
npm install
node server.js
```
